# 🏆 Álbum Digital Mundial FIFA 2026

![Android Studio](https://img.shields.io/badge/Android_Studio-3DDC84?style=for-the-badge&logo=android-studio&logoColor=white)

Una aplicación móvil para Android diseñada para que los aficionados del fútbol coleccionen estampas virtuales y físicas de los jugadores, estadios y uniformes del Mundial 2026 de la FIFA.

## 📖 Sobre el Proyecto

El concepto principal de esta aplicación es funcionar como un álbum digital interactivo. Se complementa con un prototipo de álbum físico donde los usuarios pueden adquirir estampas con diseños atractivos que incluyen un código QR en la parte posterior. Al escanear este código con la aplicación, la estampa se "reclama" y se añade a la colección digital del usuario.

**Autor:** Miguel Moreno.

## ✨ Características Principales

* **📸 Escáner con Realidad Aumentada:** Utiliza la cámara del dispositivo para escanear el reverso de las estampas físicas.
* **📚 Álbum Virtual y Búsqueda:** Visualiza todas las estampas desbloqueadas en una lista desplegable. Incluye una barra de búsqueda en tiempo real para filtrar fácilmente por nombre.
* **🔍 Detalles de Estampas:** Al hacer clic en una estampa, accede a una vista detallada con la información más relevante del jugador, estadio o uniforme, todo separado por categorías.
* **🔐 Gestión de Usuarios:** * Inicio de sesión y registro de nuevas cuentas con la opción de subir una foto de perfil.
  * Actualización de datos del usuario, incluyendo nombre(s), apellido(s), contraseña y foto de perfil.
  * Opción para cerrar sesión.

## 🏗️ Desarrollo y Tecnologías

* **Entorno de Desarrollo:** Android Studio versión JellyFish 2023.3.1.
* **Escaneo de Códigos QR:** Integración de cámara para la lectura de códigos QR en las estampas físicas.
